# CCNA Network Design & Implementation Project

## Overview
This project was designed and implemented based on CCNA objectives.
It simulates a small to medium enterprise network with secure and controlled access.

## Network Features & Objectives
- Network topology design
- IP addressing and subnetting
- Routing and switching
- VLAN configuration
- Inter-VLAN routing
- Access Control Lists (ACL)
- Network security fundamentals
- Connectivity testing and troubleshooting

## Tools & Technologies
- Cisco Packet Tracer
- Cisco Routers & Switches
- VLAN
- ACL (Standard & Extended)
- Static and Dynamic Routing

## My Responsibilities
- Designed the complete network topology
- Configured routers and switches
- Implemented VLANs and inter-VLAN routing
- Applied Access Control Lists to control traffic
- Secured the network according to CCNA objectives
- Tested and verified network connectivity
- Performed troubleshooting and optimization

## Result
The network was successfully implemented with proper access control,
secure communication, and full connectivity between authorized devices
